# RV32I Compiler

Compiles a silly made up language (c) to RV32I assembly. To be used with [rv32i-asm](https://github.com/oxrinz/rv32i-asm). Part of [rv32i](https://github.com/oxrinz/rv32i)

---

## Currently supports
- [x] Precedence climbing
- [x] Add, sub, mul, div
- [x] Bitwise ops
- [x] Logical ops
- [ ] Short-circuiting
